# Tkinter Scientific Tool GUI Template

This folder contains only a reusable GUI layout/style template.

It does not include domain-specific model code.

## Features

- Left-side parameter panel.
- Right-side plot canvas.
- Bottom result table.
- Status log.
- Run, clear, save, and open-output-folder button pattern.
- No-console `.vbs` launcher and debug `.bat` launcher.

## Requirements

```powershell
py -3 -m pip install pandas pillow
```

`tkinter` is included in the standard Windows Python installer.

## Start

Double-click:

```text
launch_gui.vbs
```

Or run:

```powershell
py -3 app.py
```
